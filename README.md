# Проверка целостности файлов (File Integrity Checker)

Этот проект реализует веб-инструмент для проверки целостности файлов (изображений или кода) с использованием алгоритмов хеширования MD5 и SHA-256.

## Функционал

- Загрузка файла с локального диска.
- Выбор алгоритма хеширования (MD5 или SHA-256).
- Расчет хеш-суммы файла.
- Возможность ввода ожидаемого хеша для сравнения.
- Визуальное отображение результата проверки (Совпадает/Не совпадает).

## Технологический стек

- HTML5
- CSS3 (Tailwind CSS для стилизации)
- JavaScript (ES6+)
- Библиотека [CryptoJS](https://github.com/brix/crypto-js) для вычисления хешей

## Структура проекта

- `index.html` - Основная разметка.
- `style.css` - Кастомные стили (дополнение к Tailwind).
- `script.js` - Логика приложения.

## Как использовать

1. Откройте файл `index.html` в современном браузере.
2. Выберите файл для проверки.
3. Выберите алгоритм (MD5 или SHA-256).
4. Нажмите кнопку "Рассчитать хеш".
5. (Опционально) Вставьте ожидаемый хеш в соответствующее поле для автоматического сравнения.
